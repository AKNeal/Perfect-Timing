⏰ Opening & Closing Routine

1Arrive by 4:45 AM to prepare before the first pickup window opens at 5:00 AM.
2Open the Dashboard tab and review all pending orders for the day.
3Pull, prep, and package all orders before their scheduled pickup windows.
4Mark each order Ready in the app as soon as it is packaged.
5After the last pickup, mark all fulfilled orders Complete and close down by 10:00 AM.

📦 Order Fulfillment

1Go to the Dashboard tab — new orders appear here automatically.
2Read the order details: customer name, items, quantities, and pickup time.
3Mark the order Prepping so the customer sees it is being worked on.
4Package each item carefully. Label the bag with the customer name and order number.
5Stage completed orders in the pickup area and mark Ready in the app.
6Hand off to the customer and mark Complete.

💡 Cash discount pricing is applied automatically at checkout — no manual adjustment needed.

🍽️ Menu & Pricing Changes

1Go to the Menu Mgmt tab in this admin panel.
2Toggle item availability on or off to reflect what is in stock today.
3To change a price, locate the item and update the price field. Save before leaving the tab.
4To add or permanently remove a menu item, contact Neal Media — new item codes must be added at the system level.

⚠️ If ownership of this location changes, notify Neal Media immediately so the account can be transferred.
